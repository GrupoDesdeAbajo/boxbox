boxbox api documentation

boxbox

global boxbox object

function createWorld

  • param1 canvas element to render on
  • param2 options object
  • option1 gravity (default 10) can be negative
  • option2 allowSleep (default true) bodies may sleep when they come to rest. a sleeping body is no longer being simulated, which can improve performance.
  • option3 scale (default 30) scale for rendering in pixels / meter
  • return a new World
without options var canvasElem = document.getElementById("myCanvas"); var world = boxbox.createWorld(canvasElem); with options var canvasElem = document.getElementById("myCanvas"); var world = boxbox.createWorld(canvasElem, {   gravity: 20,   scale: 60 });

Entity

a single physical object in the physics simulation

function applyImpulse

  • param1 power of impulse
  • param2 angle in degrees OR x of vector
  • param3 y of vector
Apply an instantanious force on this Entity

function canvasPosition

  • param unimplemented
  • return {x,y}
Get the Entity position in pixels. Useful for custom rendering. Unlike position the result is relative to the World's scale and camera position.

function clearForce

  • param name of force
Stop the force with the given name.

function clearVelocity

  • param name of velocity
Stop the constant velocity with the given name.

function destroy

destroy this entity and remove it from the world

function friction

  • param number (optional)
  • return number
get or set entity friction

function onFinishContact

  • param callback
  • callbackParam Entity contact ended with
  • callbackThis this Entity
Handle end of contact with another entity.

function onImpact

  • param callback
  • callbackParam1 the Entity collided with
  • callbackParam2 the normal force of the impact
  • callbackParam3 the tangential force of the impact
  • callbackThis this Entity
Handle impact with another entity.

function onKeydown

  • param callback
  • callbackParam keydown event
  • callbackThis this Entity
Handle keydown event for this entity.

function onKeyup

  • param callback
  • callbackParam keyup event
  • callbackThis this Entity
Handle keyup event for this entity.

function onStartContact

  • param callback
  • callbackParam Entity contact started with
  • callbackThis this Entity
Handle start of contact with another entity.

function position

  • param {x,y} (optional)
  • return {x,y}
get or set entity position

function rotation

  • param degrees (optional)
  • return degrees
get or set entity rotation

function setForce

  • param1 name of force
  • param2 power of force
  • param3 angle in degrees OR x of vector
  • param4 y of vector
Apply a constant force on this Entity. Can be removed later using clearForce.

function setVelocity

  • param1 name of velocity
  • param2 power of force
  • param3 angle in degrees OR x of vector
  • param4 y of vector
Continuously override velocity of this Entity. Can be removed later using clearVelocity.

  • return {x,y}
get or set position of camera

function createEntity

  • param options object
  • option1 name used to identify this Entity
  • option2 x starting x coordinate
  • option3 y starting y coordinate
  • option4 type 'dynamic' or 'static'. static objects can't move
  • option5 shape 'square' or 'circle' or 'polygon'
  • option6 height for box (default 1)
  • option7 width for box (default 1)
  • option8 radius for circle (default 1)
  • option9 points for polygon [{x,y}, {x,y}, {x,y}] must go clockwise must be convex
  • option10 density (default 2)
  • option11 friction (default 1)
  • option12 restitution or bounciness (default .2)
  • option13 active (default true) participates in collisions and dynamics
  • option14 fixedRotation (default false)
  • option15 bullet (default false) perform expensive continuous collision detection
  • option16 image file for rendering
  • option17 imageOffsetX (default 0) for image
  • option18 imageOffsetY (default 0) for image
  • option19 imageStretchToFit (default false) for image
  • option20 draw custom draw function, params are context, x, and y
  • return a new Entity
example var player = world.createEntity({   name: "player",   shape: "circle",   radius: 2 });

function createJoint

  • param1 type one of: distance, revolute, gear, friction, prismatic, weld, pulley, mouse, line
  • param2 Entity on one side of the joint
  • param3 Entity on the other side of the joint
  • param4 options
  • option enableMotor (default false)
Experimental joint support. See box2d documentation for more info.

function find

  • param1 x1 upper left of query box
  • param2 y1 upper left of query box
  • param3 x2 lower right of query box
  • param4 y2 lower right of query box
  • return array of Entities. may be empty
find Entities in a given query box
